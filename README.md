## Requirements

- Python >= 3.5.2
- Matplotlib >= 2.0.2
- Numpy >= 1.15.0
- Thrift >= 0.11.0

## Install
Add the following to your ~/.bashrc
`export PYTHONPATH=$PYTHONPATH:/path/you/want/to/add`

Source: https://askubuntu.com/questions/470982/how-to-add-a-python-module-to-syspath/471168
